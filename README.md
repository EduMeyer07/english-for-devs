# english-for-devs
Um aplicativo interativo e moderno projetado para ajudar desenvolvedores iniciantes a destravarem o vocabulário técnico em inglês mais utilizado no dia a dia da programação.

---

## 🚀 Sobre o Projeto

Muitas vezes, a maior barreira para quem está começando na tecnologia não é a lógica de programação, mas sim os termos em inglês usados em códigos, terminais e documentações. 

O **English for Devs** resolve isso de forma prática através de:
* **Flashcards Interativos:** Cartões com rotação 3D contendo os termos em inglês e suas traduções.
* **Exemplos no Código:** Cada termo conta com um trecho de código real mostrando como ele é aplicado na prática.
* **Dicionário com Busca Dinâmica:** Filtre por termos ou categorias específicas em tempo real.
* **Salvamento de Progresso:** Marque as palavras que você já domina e acompanhe sua evolução (os dados são salvos diretamente no seu navegador).

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído focando em performance, simplicidade e um visual moderno ("Tech Dark"):

* **HTML5** (Estrutura da página)
* **Tailwind CSS** (Estilização rápida e responsiva)
* **JavaScript Puro (ES6+)** (Manipulação do DOM, filtros, busca e persistência com `localStorage`)
* **FontAwesome** (Ícones de interface)
* **Google Fonts** (Fontes *Plus Jakarta Sans* e *JetBrains Mono*)

---

## 🧑‍💻 Como testar o projeto localmente

Se quiser rodar o projeto no seu computador:

1. Faça o download ou clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador de internet.
